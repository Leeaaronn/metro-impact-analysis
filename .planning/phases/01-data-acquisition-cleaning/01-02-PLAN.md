Create `src/clean.py` with these public functions (all with type hints):

**Function 1: `clean_ridership(input_path: str = "data/raw/ridership_monthly.csv", output_path: str = "data/clean/ridership_clean.csv") -> pd.DataFrame`**

Steps:
1. Read raw CSV with pd.read_csv
2. Print "Before cleaning: {N} rows"
3. Rename columns to snake_case: `year`, `month`, `line_name`, `day_group`, `avg_daily_boardings`, `line_category`
4. Filter to `line_category == "Train"` only (drop bus data)
5. Filter to `day_group == "Weekday"` only (primary analysis metric)
6. Create `date` column as pd.Timestamp from year + month (first day of month): `pd.to_datetime(df['year'].astype(str) + '-' + df['month'], format='%Y-%B')`
7. Map line names to a standardized `line_id` column for longitudinal consistency:
   - "Gold Line" -> "gold"
   - "L Line (Gold)" or any L Line variant -> "gold"
   - "A Line (Blue)" or any A Line variant -> "a_line" (post-Regional Connector)
   - "Blue Line" -> "blue"
   - "Red Line" or "B Line" variants -> "red_purple" (B/D share trunk)
   - "Purple Line" or "D Line" variants -> "red_purple"
   - "Green Line" or "C Line" variants -> "green"
   - "Expo Line" or "E Line" variants -> "expo"
   - "K Line" or "Crenshaw" variants -> "k_line"
8. Create `is_treatment` column: True when `line_id == "gold"` (pre-June 2023) or when `line_id == "a_line"` (post-June 2023, since A Line inherited Gold Line Foothill stations)
9. Create `period` column: "pre" if date < 2016-03-01, "post" if date >= 2016-04-01, "transition" if date is March 2016
10. Drop rows with null `date`, `line_name`, or `avg_daily_boardings`
11. Sort by date, line_name
12. Print "After cleaning: {N} rows"
13. Print "Date range: {min_date} to {max_date}"
14. Print "Lines found: {unique line_ids}"
15. Flag date gaps: for each line_id, check if months are continuous; print warnings for any gaps
16. Save to output_path as CSV with index=False
17. Return the DataFrame

**Function 2: `clean_station_locations(input_path: str = "data/raw/station_locations.csv", output_path: str = "data/clean/station_locations_clean.csv") -> pd.DataFrame`**

Steps:
1. Read raw CSV
2. Ensure column names are snake_case (should already be from acquire.py)
3. Parse `opened_date` as pd.Timestamp
4. Validate: no null station_name, line, lat, lon
5. Validate: lat values between 33.0 and 35.0 (LA County range), lon between -119.0 and -117.0
6. Print row count and number of Foothill 2016 stations
7. Save to output_path
8. Return DataFrame

**Main block**: `if __name__ == "__main__":` calls both functions and prints summary.

After creating clean.py, RUN it: `python src/clean.py`
